Proyecto 1

Decision support

Sistema de Apoyo a la Toma de Decisiones Clínicas (IA)

Proyecto 3

Automatización fullstack

SheetFlow Automation Hub

Plataforma fullstack para automatizar procesamiento de planillas XLSX con trazabilidad por corrida, aislamiento lógico por empresa y operación auditable de extremo a extremo.

Contexto: Operación con planillas heterogéneas provenientes de upload directo y URL remotas (OneDrive, SharePoint, Google Sheets).

Problema: Flujo manual frágil, baja observabilidad y dificultad para repetir resultados de manera consistente entre empresas.

Solución: Endpoint único POST /api/automation/run con backend Node como orquestador y motor Python para ejecución de negocio.

Decisiones técnicas: Separación estricta Frontend/API/Python, persistencia de corridas en PostgreSQL y normalización de errores para consumo de UI.

Seguridad y operación: Sesiones HTTP-only, CORS restringido, Helmet, control de upload XLSX y limpieza de temporales por corrida.

Impacto: Procesamiento repetible y auditable, menor fricción operativa y base preparada para escalar a colas asíncronas.

Tecnologías:
React 18 Vite Node.js Express PostgreSQL Python

Repositorio disponible en GitHub

Proyecto 4

Automatización táctica

Sistema de Reuniones - Tutorías UFT (Google Apps Script)

Aplicación web en GAS para administrar personas, disponibilidad horaria y agendamiento de reuniones con Google Meet en el Programa de Tutorías UFT.

Contexto: Coordinación manual de tutorías con poca trazabilidad, validaciones débiles y alta fricción operativa.

Problema: Agendamientos fuera de reglas (feriados, fines de semana, bloques no válidos) y riesgo de desorden en reservas.

Solución: Routing por rol, disponibilidad en Google Sheets, bloqueo de concurrencia con LockService y creación de eventos con Google Meet.

Decisiones técnicas: Arquitectura server-side en Apps Script para despliegue rápido, bajo costo y adopción inmediata por equipos académicos.

Impacto: Menor carga administrativa, mejor control de agendas y trazabilidad de cambios en el programa.

Escenario ideal: Operaciones institucionales que necesitan automatización confiable sin sobreinfraestructura.

Tecnologías:
Google Apps

Repositorio disponible en GitHub

Project 3

Fullstack automation

SheetFlow Automation Hub

Fullstack platform that automates XLSX processing with run-level traceability, logical tenant isolation, and auditable end-to-end operations.

Context: Heterogeneous spreadsheets from direct uploads and remote URLs (OneDrive, SharePoint, Google Sheets).
