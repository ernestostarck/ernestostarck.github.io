Proyecto 1

Decision support

Sistema de Apoyo a la Toma de Decisiones Clínicas (IA)

Proyecto de título con validación en contexto clínico real, orientado a apoyar criterio profesional bajo estándares de uso responsable.

Contexto: Centro Avanzado de Tratamiento de Heridas, con necesidad de estandarizar evaluaciones periódicas y continuidad de seguimiento.

Problema: Alta variabilidad entre evaluaciones clínicas y dependencia de experiencia individual para decisiones reiteradas.

Solución: Sistema de apoyo con IA aplicada para estructurar información clínica y entregar sugerencias sin reemplazar juicio profesional.

Decisiones técnicas: Arquitectura modular, persistencia local controlada y separación estricta entre lógica de IA y capa de presentación.

Validación: Prototipo académico validado en contexto clínico real, con énfasis en uso responsable y apoyo al criterio profesional.

Impacto: Mayor consistencia de evaluación y mejor continuidad del seguimiento clínico.

Tecnologías:
React Node.js Express SQLite Kotlin

Proyecto 2

Sistema productivo

Sistema de Gestión y Modernización para APR

Modernización operativa en contexto de baja madurez tecnológica, priorizando trazabilidad y mantenibilidad por sobre complejidad innecesaria.

Contexto: Procesos administrativos y contables manuales, apoyados en registros físicos y planillas no integradas.

Problema: Cálculos manuales de cobro, baja trazabilidad y alto costo de tiempo para auditoría histórica.

Solución: Sistema web para centralizar información, automatizar procesos críticos y reducir errores operativos.

Decisiones técnicas: Arquitectura desacoplada para mantenimiento, consultas flexibles y diseño de interfaz para usuarios no técnicos.

Consideraciones: Trade-off explícito a favor de simplicidad operativa y continuidad del servicio.

Impacto: Menor carga administrativa, mejor consistencia de datos y base sólida para escalabilidad futura.

Tecnologías:
React JavaScript GraphQL Docker

Repositorio disponible en GitHub

Proyecto 3

Integración corporativa

PowerPoint Add-In para Automatización de Reportes Corporativos

Automatización en entorno corporativo con sistemas legacy, enfocada en adopción real y reducción de fricción organizacional.

Contexto: Reportes ejecutivos en PowerPoint alimentados manualmente desde múltiples fuentes empresariales.

Problema: Inconsistencias de datos, retrabajo reiterado y dependencia operativa del usuario final.

Solución: Add-In integrado en PowerPoint para actualización directa de contenido desde fuentes corporativas.

Decisiones técnicas: Integración nativa con Office, separación de lógica de negocio y compatibilidad con infraestructura existente.

Consideraciones: Se priorizó adopción y bajo costo de cambio frente a reestructuraciones disruptivas.

Impacto: Menos errores manuales, ahorro de tiempo y mayor consistencia en información ejecutiva.

Tecnologías:
C# VSTO Oracle

Repositorio disponible en GitHub

Proyecto 4

Automatización táctica

Sistema de Coordinación de Reuniones (Google Apps Script)

Solución liviana para eliminar tareas repetitivas y reducir fricción administrativa sin agregar infraestructura.

Contexto: Programas de tutorías con coordinación manual de reuniones y seguimiento de participantes.

Problema: Tareas repetitivas, validaciones ausentes y pérdida de tiempo operativo.

Solución: Automatización con Google Apps Script para orquestar flujos y reglas de coordinación.

Decisiones técnicas: Herramienta liviana y de rápida adopción, evitando sobrearquitectura.

Impacto: Menor carga administrativa y mejor organización de tutorías.

Escenario ideal: Procesos de bajo costo donde velocidad de implementación es prioridad.

Tecnologías:
Google Apps

Repositorio disponible en GitHub
